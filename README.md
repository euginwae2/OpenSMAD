## ğŸš€ Getting Started

### Prerequisites

- Python 3.9+
- Node.js 16+ (for frontend components)
- Docker (optional, for containerized deployment)

### Installation

```bash
# Clone the repository
git clone https://github.com/opensmad/opensmad.git
cd opensmad

# Install backend dependencies
pip install -r requirements.txt

# Install frontend dependencies
cd frontend
npm install

# Run development servers
# Backend (API & services)
python run.py

# Frontend (UI)
npm run dev
```

### Quick Start Guide

1. **Sign up / Log in** via the web interface.
2. **Choose a template** (e.g., Lunar CubeSat, LEO Constellation) or start from scratch.
3. **Explore the Dashboard** â€“ edit parameters and watch the mission update in real time.
4. **Run a Trade Study** â€“ vary inputs like altitude or mass to optimize your design.
